## New Features

### Interactive Content Customization
- Choose audience level (beginner/intermediate/advanced)
- Select teaching style (progressive/direct/reference)
- Toggle content elements (code, diagrams, tables, analogies)
- Use CLI flags for non-interactive mode

### Automatic Mermaid Conversion
- Real-time PNG conversion during generation
- LLM-based fixing for broken diagrams
- Styled diagrams with emojis and color coding
- Standalone CLI command for post-processing
